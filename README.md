##  Getting Started

###  Prerequisites

Before getting started with Pokedex, ensure your runtime environment meets the following requirements:

- **Programming Language:** Python
- **Package Manager:** Npm


###  Installation

Install Pokedex using one of the following methods:

**Build from source:**

1. Clone the Pokedex repository:
```sh
❯ git clone https://github.com/tomimoder/Pokedex
```

2. Navigate to the project directory:
```sh
❯ cd Pokedex
```

3. Install the project dependencies:


**Using `npm`** &nbsp; [<img align="center" src="" />]()

```sh
❯ echo 'npm install'
```




###  Usage
Run Pokedex using the following command:
**Using `npm`** &nbsp; [<img align="center" src="" />]()


Frontend
```sh
❯ echo 'npm run dev'
```

Backend
```sh
❯ echo 'python manage.py runserver'
```
